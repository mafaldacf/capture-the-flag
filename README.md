# Capture The Flag

```zsh
javac CaptureTheFlag.java

# Usage:
# - GRID_SIZE >= 10
# - NUM_BLUE_AGENTS <= GRID_SIZE
# - NUM_RED_AGENTS <= GRID_SIZE
# - FLAG_SPAWN_TYPE is A, B or C
java CaptureTheFlag GRID_SIZE NUM_BLUE_AGENTS NUM_RED_AGENTS FLAG_SPAWN_TYPE

# e.g.
java CaptureTheFlag 50 28 10 C
```
